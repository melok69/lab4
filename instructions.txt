# Инструкции по запуску проекта

# 1. Убедитесь, что установлен Python 3.10 или новее.
# 2. Установите зависимости, используя команду:
#    pip install -r requirements.txt

# 3. Создайте токен для вашего Telegram-бота и замените строку YOUR_TELEGRAM_BOT_TOKEN в файле telegram_bot.py.

# 4. Запустите проект:
#    - Для запуска Telegram-бота используйте:
#      python telegram_bot.py
#    - Для запуска графического интерфейса используйте:
#      python main.py
